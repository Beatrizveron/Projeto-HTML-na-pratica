Para definir uma quebra temática em uma página Html, utiliza-se a tag <hr>. Essa quebra temática serve para separar os conteúdos de uma página, normalmente é exibida como uma espécie de régua/linha na página.

Quebra de linhas

Para definir uma quebra de linha em uma página Html, utiliza-se a tag <br>. Essa quebra de linha serve para separar usar uma nova linha sem iniciar outro parágrafo.


<p>Este <br> é um <br> Parágrafo <br> com <br> quebra <br> de <br> linha </p>

Nota: a tag <br> é uma tag vazia então não pode possuir tag final </>

Poema

Se você colocar o texto de uma poema com o elemento <p>, ele será exibido em uma única linha. A solução para que o poema fique com o texto pré-formatado, é utilizar o elemento <pre>, assim o texto será exibido com uma fonte de largura fixa e preservará espaços e quebras de linha.


Nota: o elemento <pre> não é uma tag vazia então deve possuir tag final </pre>
