Introdução html

O HTML ou Hyper Text Markup Language, é uma linguagem de marcação padrão utilizada para desenvolver e criar páginas web.

Como funciona?

O HTML descreve toda a estrutura básica de uma página web;

O Html se constitui em muitos elementos. Tais elementos declaram ao navegador em como deve-se exibir o conteúdo da página;

Os elementos HTML reconhecem partes de conteúdos como cabeçalho (header), rodapé(footer), parágrafo(p), etc.

O que é um elemento?

Todo elemento HTML é definido por uma tag inicial e tag final. Entre o começo e fim da tag, encontra-se o conteúdo. Logo tudo que encontra-se desde a tag inicial e final, é um elemento HTML.

Documento HTML simples

Explicação:

<!DOCTYPE html>: declara que o documento é um documento HTML5;

<html>: Caracterizado por ser a raiz de uma página web;

<head>: Esse elemento mantém informações gerais (metadados) sobre a página, como seu título, links para scripts, etc;

<title>: É nesse elemnto que informamos um título para a página web, esse título é mostrado na barra de título do navegador ou na guia da página;

<body>: Nesse elemento define-se o corpo do documento, ou seja, todo conteúdo visível. Exemplo: Tabelas, imagens, parágrafos, etc;

<p>: elemento que define todo início de parágrafo.
